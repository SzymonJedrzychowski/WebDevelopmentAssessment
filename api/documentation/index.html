Endpoints

/api

Description: Returns student data

Example: http://unn-w20020581.newnumyspace.co.uk/year3/week5/api

Responses:

200 - Successful return

405 - Incorrect method

/actors

Description: Returns data of actors

Example: http://unn-w20020581.newnumyspace.co.uk/year3/week5/actors

Parameters:

(optional) film_id: int - id of movie

Description: Returns data of actors that played in movie with given id.

Example: http://unn-w20020581.newnumyspace.co.uk/year3/week5/actors?film_id=1

Responses:

200 - Successful return

400 - Partially successful return - at least one of paramethers was incorrect

405 - Incorrect method

/films

Description: Returns data of film

Example: http://unn-w20020581.newnumyspace.co.uk/year3/week5/films

Parameters:

(optional) language: string - name of language

Description: Returns movies with given language.

Example: http://unn-w20020581.newnumyspace.co.uk/year3/week5/films?language=french

Responses:

200 - Successful return

400 - Partially successful return - at least one of paramethers was incorrect

405 - Incorrect method
